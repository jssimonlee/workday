# 🗓️ 2개월 근무일 계산기 (2-Month Workday Calculator)

정직원(기준 근무일) 및 공무직 근로자의 2개월 단위 총 근무일 수를 정확하고 편리하게 계산해 주는 웹 기반 애플리케이션입니다. 

## ✨ 주요 기능
- **2개월 단위 근무일 계산**: 선택한 연도와 2개월(짝)을 기준으로 전체 근무일을 자동 계산합니다.
- **직군별 맞춤형 근무 로직**:
  - **기준 근무일**: 평일(월~금) 5일 근무 기준
  - **공무직**: 특정 평일 1일 + 주말 1일 휴무, 그리고 소속 도서관의 휴관일(1,3주차 또는 2,4주차 월요일 등)을 반영하여 복합 계산
- **도서관 휴관일 이월 로직**: 매월 첫 주가 수요일 등으로 시작할 경우, 전월의 휴일 일정을 따르는 복잡한 주 단위 휴관일 로직을 정확히 판별합니다.
- **휴일 커스텀 관리**:
  - 기본 공휴일 외에 개인적으로 휴일(대체 휴일 등)을 손쉽게 추가 및 삭제할 수 있습니다.
  - 사용자가 설정한 휴일 데이터는 브라우저의 `LocalStorage`에 안전하게 보관되어 새로고침해도 유지됩니다.
- **반응형 웹 디자인**: PC뿐만 아니라 모바일 환경(카카오톡 인앱 브라우저 등)에서도 깨짐 없이 완벽하게 동작하는 유연한 UI/UX를 제공합니다.

## 🚀 사용 방법
별도의 설치나 서버 구축 없이 즉시 사용할 수 있습니다.

1. 저장소를 클론(Clone)하거나 다운로드합니다.
2. `index.html` 파일을 더블 클릭하여 웹 브라우저에서 엽니다.
   - (또는 GitHub Pages 기능을 켜서 바로 웹 주소로 접속할 수도 있습니다.)
3. 왼쪽 설정 탭에서 대상 월, 도서관 휴관일, 개인 휴일 등을 설정합니다.
4. **[공무직 휴관일 달력 확인하기]** 버튼을 누르면 2개월간의 캘린더와 최종 근무일 수가 산출됩니다.

## 🛠️ 기술 스택
- **HTML5 / CSS3 / Vanilla JavaScript**: 외부 서버에 의존하지 않는 100% 클라이언트 사이드 로직
- **Tailwind CSS (CDN)**: 모던하고 예쁜 파스텔 톤의 반응형 UI 구현

## 📂 파일 구조
- `index.html`: 메인 레이아웃 및 팝업 모달 UI
- `styles.css`: 추가 커스텀 디자인 및 스크롤바 스타일링
- `script.js`: 날짜 계산, 캘린더 렌더링, LocalStorage 연동 및 이벤트 처리 로직
- `holidays.js`: 매년 변동되는 국가 공휴일 목록 데이터 관리 (직접 수정 가능)

## 💡 유지보수 안내
- 매년 새해의 법정 공휴일이나 선거일 등은 `holidays.js` 배열에 날짜(`YYYY-MM-DD`)와 이름만 추가/수정하여 GitHub에 Push 하면 즉시 반영됩니다.
